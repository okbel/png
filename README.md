## Ejemplos:

### Minimal
Contiene lo minimo para empezar un server con GraphQL usando Express

1. Corré: `npm run minimal` o `yarn minimal`
2. El servidor ya está corriendo en localhost. `http://localhost:3000/?query=%7B%0A%20%20hello%0A%7D`
